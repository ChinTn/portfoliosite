# Personal Portfolio Website

A full-stack portfolio website built using the MERN stack to showcase my projects, blogs, skills, and experiences.

## Tech Stack

- React.js
- Tailwind CSS
- Node.js
- Express.js
- MongoDB

## Features

- Responsive UI
- Dynamic project & blog management
- Contact form integration
- Resume download
- Full-stack architecture

---

Built with the MERN stack.
